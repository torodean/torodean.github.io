Information

The best way to learn coding is by coding. MIA (Multiple Integrated Applications) is a program where I can store useful coding functions I've created in one place for easy accessibility. It is a project I've started to better improve my programming skills and give me something to work on in my free time.

Features

MIA is a cross platform terminal program that runs on both Windows and Linux. MIA contains various random features. MIA is very new and a lot of already coded (as part of other programs) features are still in the process of being integrated with MIA. Current features included in MIA include but are not limited to the following.

  • Basic string encryption through a custom algorithm with upcoming advanced file encryption in development.
  • Various mathematical functions such as add/subtract/multiply functions for integers of any length.
  • Prime number generation and ordered prime number list generation.
  • Windows key code simulation with numerous Minecraft and WoW features for performing tasks while afk.
  • Comprehensive workout generation program.
  • More and more to come...
