Information

Antonius's Handbook is a collection of useful tools. There are currently two volumes for Antonius' Handbook. Mathematial Mansian (Volume I) contains formulas, constants, units and definitions pertaining to mathematics and physics. Programmers Paradise (Volume II) contains useful coding features, code snippets, and programming tools for various operating systems and computer languages.

It’s purpose is to provide a complete ’encyclopedia’ per say of various useful and significant ideas used often. The idea behind it is to be a quick reference providing easily accessible access to necessary information for either double checking or recalling proper formula or syntax for use in various situations.
